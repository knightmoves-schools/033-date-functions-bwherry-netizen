# 033 Date Functions

[![033 Date Functions](https://img.youtube.com/vi/niqXiGF1hGQ/0.jpg)](https://www.youtube.com/watch?v=niqXiGF1hGQ)

The SQL in the `exercise.sql` file:
- Should return all columns and rows from `Grocery` where the product's `NEXT_EXPIRE_DATE` is on or before today's date.
- Should contain the `DATE()` function

Additional Info:
- When you begin this assignment, `NEXT_EXPIRE_DATE`s in `Grocer` may be out dated, so that all the dates are prior to the current date. Autograding the assignment will update all `NEXT_EXPIRE_DATE` values in the table so only a subset of the dates are prior to the current date, depending on the date you complete this assignment. Regardless of what is in the table prior to submitting the assignment, you should complete the assignment per the instructions above.
